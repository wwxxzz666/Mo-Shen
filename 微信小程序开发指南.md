## 🔍 项目分析

### 当前项目状态
- **项目名称**：墨神 Mo-Shen (StoryAgents)
- **项目类型**：多智能体小说创作工作台
- **技术栈**：
  - 后端：Python + LangGraph（多智能体编排）
  - 前端：纯 HTML/CSS/JavaScript（Web 页面）
  - 服务器：Python HTTP 服务器

### 小程序化挑战
| 挑战项 | 当前状态 | 小程序要求 |
|--------|----------|-----------|
| 项目结构 | Web 项目 | 需要 app.json/app.js/app.wxss |
| 前端技术 | HTML/CSS/JS | WXML/WXSS/JS + WXS |
| API 请求 | HTTP | HTTPS（必须） |
| 流式响应 | Fetch API | wx.request 或 WebSocket |
| 文件导出 | docx.js（浏览器） | 云端生成 + 下载 |
| 本地存储 | localStorage | wx.setStorage/getStorage |

### Step 2: 支持流式响应（WebSocket）

小程序不支持 Fetch EventSource，需要使用 WebSocket：

```python
from fastapi import WebSocket

@app.websocket("/ws/story/{story_id}")
async def websocket_endpoint(websocket: WebSocket, story_id: str):
    await websocket.accept()
    try:
        while True:
            # 接收小程序消息
            data = await websocket.receive_json()
            
            # 执行创作任务，流式返回进度
            async for update in run_story_workflow_streaming(...):
                await websocket.send_json({
                    "type": "progress",
                    "agent": update["agent"],
                    "content": update["content"]
                })
            
            # 完成后通知
            await websocket.send_json({
                "type": "complete",
                "story_id": story_id
            })
    except Exception as e:
        await websocket.send_json({"type": "error", "message": str(e)})
```

### Step 2: API 工具层 (utils/api.js)

```javascript
/**
 * StoryAgents API 封装
 */

const BASE_URL = 'https://your-domain.com/api'; // 替换为你的域名

/**
 * 发起 HTTPS 请求
 */
function request(url, method, data) {
  return new Promise((resolve, reject) => {
    wx.request({
      url: `${BASE_URL}${url}`,
      method: method,
      data: data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': wx.getStorageSync('token') || ''
      },
      success: (res) => {
        if (res.statusCode === 200) {
          resolve(res.data);
        } else {
          reject(res.data);
        }
      },
      fail: (err) => {
        reject(err);
      }
    });
  });
}

/**
 * 创建故事项目
 */
function createStory(data) {
  return request('/stories', 'POST', data);
}

/**
 * 获取故事状态
 */
function getStoryStatus(storyId) {
  return request(`/stories/${storyId}/status`, 'GET');
}

/**
 * 生成章节
 */
function generateChapter(storyId, data) {
  return request(`/stories/${storyId}/chapters`, 'POST', data);
}

/**
 * 导出故事
 */
function exportStory(storyId, format = 'docx') {
  return request(`/stories/${storyId}/export?format=${format}`, 'GET');
}

module.exports = {
  createStory,
  getStoryStatus,
  generateChapter,
  exportStory
};
```

### Step 3: WebSocket 管理 (utils/websocket.js)

```javascript
/**
 * WebSocket 连接管理（用于流式接收创作进度）
 */

let socketTask = null;
let isConnected = false;

/**
 * 建立 WebSocket 连接
 */
function connect(storyId, onMessage, onError) {
  if (isConnected) {
    disconnect();
  }

  socketTask = wx.connectSocket({
    url: `wss://your-domain.com/ws/story/${storyId}`
  });

  socketTask.onOpen(() => {
    console.log('WebSocket 连接成功');
    isConnected = true;
  });

  socketTask.onMessage((res) => {
    const data = JSON.parse(res.data);
    onMessage(data);
  });

  socketTask.onError((err) => {
    console.error('WebSocket 错误:', err);
    isConnected = false;
    if (onError) onError(err);
  });

  socketTask.onClose(() => {
    console.log('WebSocket 连接关闭');
    isConnected = false;
  });

  return socketTask;
}

/**
 * 发送消息
 */
function send(data) {
  if (isConnected && socketTask) {
    socketTask.send({
      data: JSON.stringify(data)
    });
  }
}

/**
 * 断开连接
 */
function disconnect() {
  if (socketTask) {
    socketTask.close();
    socketTask = null;
    isConnected = false;
  }
}

module.exports = {
  connect,
  send,
  disconnect
};
```

### Step 5: 创作工作台逻辑 (pages/studio/studio.js)

```javascript
const websocket = require('../../utils/websocket.js');
const api = require('../../utils/api.js');

Page({
  data: {
    storyId: '',
    storyTitle: '',
    isRunning: false,
    agents: [
      { name: '策划编辑', status: 'pending', statusText: '等待中', progress: 0, output: '' },
      { name: '世界观构建', status: 'pending', statusText: '等待中', progress: 0, output: '' },
      { name: '角色设计', status: 'pending', statusText: '等待中', progress: 0, output: '' },
      { name: '大纲规划', status: 'pending', statusText: '等待中', progress: 0, output: '' },
      { name: '章节写作', status: 'pending', statusText: '等待中', progress: 0, output: '' },
      { name: '审校', status: 'pending', statusText: '等待中', progress: 0, output: '' }
    ]
  },

  onLoad(options) {
    this.setData({
      storyId: options.storyId,
      storyTitle: options.title
    });
  },

  /**
   * 开始创作
   */
  startCreation() {
    this.setData({ isRunning: true });

    // 建立 WebSocket 连接，接收实时进度
    websocket.connect(this.data.storyId, (data) => {
      this.handleWebSocketMessage(data);
    }, (err) => {
      wx.showToast({
        title: '连接失败',
        icon: 'none'
      });
      this.setData({ isRunning: false });
    });

    // 发送开始创作指令
    api.createStory({
      story_id: this.data.storyId
    }).then(res => {
      console.log('创作任务已启动', res);
    }).catch(err => {
      wx.showToast({
        title: '启动失败',
        icon: 'none'
      });
      this.setData({ isRunning: false });
    });
  },

  /**
   * 处理 WebSocket 消息
   */
  handleWebSocketMessage(data) {
    if (data.type === 'progress') {
      // 更新对应智能体的进度
      const agents = this.data.agents;
      const agentIndex = agents.findIndex(a => a.name === data.agent);
      if (agentIndex !== -1) {
        agents[agentIndex].status = 'running';
        agents[agentIndex].statusText = '进行中';
        agents[agentIndex].progress = data.progress || 50;
        agents[agentIndex].output = data.content;
        this.setData({ agents });
      }
    } else if (data.type === 'complete') {
      // 创作完成
      wx.showToast({
        title: '创作完成！',
        icon: 'success'
      });
      this.setData({ isRunning: false });
      websocket.disconnect();
    } else if (data.type === 'error') {
      wx.showToast({
        title: data.message,
        icon: 'none'
      });
      this.setData({ isRunning: false });
    }
  },

  /**
   * 暂停创作
   */
  pauseCreation() {
    // 实现暂停逻辑
    websocket.send({ action: 'pause' });
  },

  onUnload() {
    // 页面卸载时断开 WebSocket
    websocket.disconnect();
  }
});
```

## 🎯 关键技术难点解决方案

### 1. 流式响应问题

**问题**：小程序不支持 Fetch EventSource，无法接收流式响应

**解决方案**：使用 WebSocket

```javascript
// 后端（FastAPI）
from fastapi import WebSocket

@app.websocket("/ws/story/{story_id}")
async def websocket_endpoint(websocket: WebSocket, story_id: str):
    await websocket.accept()
    # 流式返回创作进度
    async for update in generate_story(story_id):
        await websocket.send_json(update)

// 前端（小程序）
wx.connectSocket({
  url: `wss://your-domain.com/ws/story/${storyId}`
});
```

### 2. 文件导出问题

**问题**：小程序无法使用 docx.js 等浏览器库

**解决方案**：服务器端生成，提供下载链接

```python
# 后端
from docx import Document
import os

@app.get("/api/stories/{story_id}/export")
async def export_story(story_id: str):
    # 生成 DOCX 文件
    doc = Document()
    doc.add_paragraph(story_content)
    file_path = f"/tmp/{story_id}.docx"
    doc.save(file_path)
    
    # 返回下载链接（实际项目中应上传到云存储）
    download_url = f"https://your-domain.com/downloads/{story_id}.docx"
    return {"download_url": download_url}
```

```javascript
// 前端
wx.downloadFile({
  url: downloadUrl,
  success: (res) => {
    wx.openDocument({
      filePath: res.tempFilePath,
      showMenu: true
    });
  }
});
```

### 3. 域名白名单问题

**问题**：小程序只能请求 HTTPS 域名，且必须在白名单中

**解决方案**：
1. 后端必须部署到支持 HTTPS 的服务器
2. 在微信公众平台配置服务器域名白名单
3. 开发环境下可以使用「不校验合法域名」选项（仅开发阶段）

### 4. 用户登录问题

**问题**：需要识别用户身份

**解决方案**：使用微信登录

```javascript
// 前端获取用户 OpenID
wx.login({
  success: (res) => {
    const code = res.code;
    // 发送 code 到后端，换取 OpenID
    wx.request({
      url: 'https://your-domain.com/api/auth/login',
      method: 'POST',
      data: { code: code },
      success: (res) => {
        wx.setStorageSync('token', res.data.token);
      }
    });
  }
});
```

```python
# 后端（使用微信接口换取 OpenID）
import requests

@app.post("/api/auth/login")
async def login(data: dict):
    code = data['code']
    appid = "your-appid"
    secret = "your-secret"
    
    url = f"https://api.weixin.qq.com/sns/jscode2session?appid={appid}&secret={secret}&js_code={code}&grant_type=authorization_code"
    res = requests.get(url).json()
    
    openid = res['openid']
    # 生成自定义登录态（JWT）
```

## 🚀 部署与发布

### 1. 后端部署检查清单

- [ ] 服务器已配置 HTTPS
- [ ] 域名已备案（如果服务器在国内）
- [ ] 微信小程序域名白名单已配置
- [ ] API 接口已测试通过
- [ ] WebSocket 连接正常
- [ ] 环境变量已配置（API Key 等）
- [ ] 错误日志已配置

### 2. 小程序发布流程

#### Step 1: 上传代码
1. 在微信开发者工具中点击「上传」
2. 填写版本号和项目备注
3. 上传成功后会生成开发版本

#### Step 2: 提交审核
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「版本管理」
3. 将开发版本提交审核
4. 填写审核信息：
   - **服务类目**：生活服务 > 综合
   - **标签**：AI创作、小说、写作辅助
   - **测试账号**：提供测试账号密码（如果需要登录）
   - **审核说明**：简要说明功能和使用场景

#### Step 3: 等待审核
- 审核时间：通常 1-3 个工作日
- 审核结果会通过微信通知

#### Step 4: 发布上线
- 审核通过后，在版本管理中点击「发布」
- 发布后立即生效

### 3. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 审核被拒：功能不完整 | 提供的测试账号无法体验完整功能 | 提供完整的测试账号，或录制演示视频 |
| 审核被拒：诱导分享 | 小程序内有强制分享才能继续的按钮 | 移除强制分享逻辑 |
| 审核被拒：类目不符 | 选择的类目与实际功能不符 | 修改为正确的类目 |
| 无法请求 API | 域名不在白名单中 | 在公众平台配置服务器域名 |
| WebSocket 连接失败 | 未使用 wss 协议 | 必须使用 wss:// 协议 |
